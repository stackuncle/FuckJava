# 题目描述
> 请编写一个函数，将两个数字相加。不得使用+或其他算数运算符。

> 给定两个int A和B。请返回A＋B的值


```java
import java.util.*;

public class UnusualAdd {
    public int addAB(int A, int B) {
        // write code here
        if(A==0)	return B;
        else if(B==0)	return A;
        else {
            return addAB(A^B,(A&B)<<1);
        }
    }
}

```
