# 题目描述
> 求出任意非负整数区间中1出现的次数（从1 到 n 中1出现的次数）。

## 思路1
数字转为字符串处理。

```java
public class Solution {
    public int countN(int x) {
        char c[] = String.valueOf(x).toCharArray();
        int count = 0;
        for (int i = 0; i < c.length; i++) {
            if (c[i] == '1') {
                count ++;
            }
        }
        
        return count;
    }
    public int NumberOf1Between1AndN_Solution(int n) {
        int res = 0;
        for(int i = 1; i <= n; i++) {
            res += countN(i);
        }
        
        return res;
    }
}
```
