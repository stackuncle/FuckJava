# 题目描述
> 给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        if(s == null || s.length() == 0) return 0;
        char[] S = s.toCharArray();
        int[] map = new int[256];
        for(int i = 0; i < 256; i++) {
            map[i] = -1;
        }
        
        int len = 0;
        int pre = -1; 
        int cur = 0;
        for(int i = 0; i < S.length; i++) {
            pre = Math.max(pre, map[S[i]]);
            cur = i - pre;
            len = Math.max(cur, len);
            map[S[i]] = i;
        }
        return len;
    }
}
```
